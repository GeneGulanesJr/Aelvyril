Three Pillars

Ward. Cipher. Recall.

Aelvyril protects the full request-response loop: sensitive text is detected, masked before transit, then restored only on your machine.

Ward

Monitors clipboard and browser inputs for sensitive patterns in real time.

Cipher

Replaces PII with context-aware tokens before any data leaves your machine.

Recall

Rehydrates the LLM response with real values, seamlessly and automatically.

Vigil

Watches for secrets, API keys, and credentials in your clipboard.

Veil

Context-aware tokenization via NER for names, locations, orgs.

Conduit

Works with any LLM — OpenAI, Anthropic, Gemini, local models.

Stronghold

OS keychain storage. No keys in config, logs, or dumps.

Local Execution

Runs on your machine. No external deps.

No Data Stored

Nothing logged. Token maps live in memory only.

No Cloud

Works offline. No server transit.

Open Source

Audit the code. Run your own build.
