Privacy Notice

Last updated: 2026-04-23

Aelvyril is built to run locally and minimize data exposure when you use AI services.

What Aelvyril Processes

  • Prompt and response text required to detect, mask, and restore sensitive values.
  • Locally generated token mappings used during a single request/response flow.
  • Configuration and app settings stored on your machine.

What Aelvyril Does Not Do

  • Does not run a hosted Aelvyril cloud service.
  • Does not intentionally store raw sensitive values beyond active local processing.
  • Does not intentionally send unmasked sensitive values to third-party LLM providers.

Third-Party Services

Your selected model provider (for example OpenAI, Anthropic, Gemini, or local endpoints) receives only the masked payload generated by Aelvyril. Their own terms and privacy policies still apply.

Analytics on Landing Page

The public landing page may use basic analytics scripts to understand page usage trends. This is separate from the local runtime behavior of the desktop application.

Contact

Questions can be raised via the Aelvyril GitHub issue tracker.
